# express-demo
学习 node、express，先撸了一个上传文件的 demo
a demo about upload/download file  with express&multer
# Quick start
+ Clone the repository :
    `git clone git@github.com:xinx1n/express-demo.git`
+ Install dependencies :
    ```
    cd express-demo
    npm install
    ```

+ Start a server :
    `node app.js`
